## Fill Out Release Form on GitHub

### 1. Tag
- **Enter**: `v1.0.0` (with the 'v' prefix, as GitHub suggests)
- Or: `1.0.0` (without 'v' - also works)
- ✅ GitHub will create this tag when you publish

### 2. Target
- **Select**: `main` (should already be selected)
- ✅ This is the branch your code is on

### 3. Release Title
- **Enter**: `ProjectCoachAI Forge Edition V1 - Initial Release`
- Or keep it simple: `V1.0.0 - Initial Release`

### 4. Description / Release Notes

Copy this text into the description field:

```markdown
# ProjectCoachAI Forge Edition V1 - Initial Release

## What's New

🎉 **Initial Release** - ProjectCoachAI Forge Edition V1 is now available!

### Features

- ✅ **Multi-Pane AI Workspace** - Compare ChatGPT, Claude, Gemini, Perplexity, Grok, DeepSeek & more side-by-side
- ✅ **AI-Powered Synthesis** - 7 analysis frameworks to synthesize insights from multiple AI responses
- ✅ **Clean, Beautiful Interface** - Modern, intuitive design built for productivity
- ✅ **Privacy-First** - All data stays local on your device (Swiss Privacy Edition)
- ✅ **Click-and-Play Installation** - Simple drag-and-drop (macOS) or installer wizard (Windows)
- ✅ **Free Forever** - No subscription required for core features

### System Requirements

- **macOS**: 10.13 or later
- **Windows**: Windows 10 or later
- **Disk Space**: ~250 MB

### Installation

1. Download the installer for your platform
2. **macOS**: Double-click `.dmg` → Drag app to Applications folder
3. **Windows**: Double-click `.exe` → Follow installer wizard
4. Launch ProjectCoachAI Forge Edition V1

### What's Included

- Multi-pane comparison workspace
- Support for 7+ AI tools
- 7 synthesis analysis frameworks:
  - Comprehensive Analysis
  - Executive Summary
  - Consensus Mapping
  - Divergence Analysis
  - Quality Assessment
  - Improvement Recommendations
  - Best-of-Best Synthesis

### Getting Started

1. Launch the app
2. Select AI tools from the toolshelf
3. Start comparing responses side-by-side
4. Use synthesis to get deeper insights

### Support

- Website: https://projectcoachai.com
- Issues: [Report issues here](https://github.com/ProjectCoachAI/projectcoachai-forge-edition/issues)

---

**Ready to launch your AI projects the smart way!** 🚀
```

### 5. Attach Binaries

**Drag and drop or click to upload:**

1. **macOS Installer:**
   - File: `dist/ProjectCoachAI Forge Edition V1-1.0.0.dmg`
   - Drag it to the "Attach binaries" section
   - Wait for upload to complete (shows file size/progress)

2. **Windows Installer:**
   - File: `dist/win/ProjectCoachAI Forge Edition V1 Setup 1.0.0.exe`
   - Drag it to the "Attach binaries" section
   - Wait for upload to complete (shows file size/progress)

**Note:** Upload can take a few minutes (files are ~150-250 MB each)

### 6. Pre-Release Checkbox

- ✅ **Uncheck** "Set as a pre-release" (since this is the official V1.0.0 release)
- Leave it unchecked for production releases

### 7. Publish

- Click the green **"Publish release"** button
- Wait for release to be created
- You'll be redirected to the release page
