Quick Start

Go from app open to first decision in one action.

  • Pick your AI perspectives and ask a question
  • Load prompts from the Prompt Library
  • Use shortcuts: C (compare), R (rank), S (synthesize)
🤖

Compare

See where AI agrees — and where it doesn't.

  • Mix Claude, ChatGPT, Gemini, and more in one workspace
  • Send one question to all AIs simultaneously
  • Spot differences at a glance in side-by-side view

Used in: Compare → Rank → Synthesize

🏆

Rank

Decide faster when multiple answers exist.

  • Lock criteria so every comparison uses the same scoring
  • Score responses in table or radar view
  • Save the winner with a clear reason why

Used in: Compare → Rank → Synthesize

🧠

Synthesize

Get one decision-ready answer from all AI perspectives.

  • Choose consensus, best-of, or hybrid synthesis
  • Export to Markdown, PDF, or clipboard
  • Use command prompts to enforce brevity

Used in: Compare → Rank → Synthesize

Prompt Library

  • Save and reuse your own prompt templates
  • Apply modifiers like "List only" or "One sentence"
