# Anthropic API Credits FAQ

## 💰 Credits and API Keys Relationship

### Key Facts:
- ✅ **Credits are tied to your ACCOUNT, not individual API keys**
- ✅ **Deactivating a key does NOT remove your credits/balance**
- ✅ **New API keys can use existing account balance**
- ✅ **Your funds are safe and remain in your account**

---

## 🔍 Check Your Account Balance

### Step 1: View Your Dashboard
1. Go to: https://console.anthropic.com/
2. Log in to your account
3. Check the **Billing** or **Credits** section
4. View your current balance

### Step 2: Verify Your Balance
- Your balance should show the same amount as before the key deactivation
- Credits are NOT tied to specific API keys
- All keys under your account share the same balance

---

## 📋 Creating a New API Key

### If Anthropic Asks You to Buy Credits:

**Option A: You Have Credits**
- You can **skip** the "Buy Credits" prompt
- Simply create the key
- Your existing balance will be available for the new key

**Option B: You Need Credits**
- If your balance is low or zero, you'll need to add credits
- Purchase the minimum amount required
- The new key will use these credits

**Option C: Minimum Balance Requirement**
- Some API providers require a minimum balance
- You may need to add credits even if you have some balance
- Check Anthropic's current minimum requirements

---

## 🔄 What Happens to Old Credits

### When an API Key is Deactivated:
1. ✅ Your account balance **remains unchanged**
2. ✅ Credits are **NOT lost or removed**
3. ✅ Credits are **available for new API keys**
4. ✅ You can continue using your existing balance

### Example:
- **Before**: You had $100 in credits + API Key A (deactivated)
- **After**: You still have $100 in credits + can create API Key B
- **Result**: New API Key B uses your existing $100 balance

---

## ⚠️ Important Notes

### Credit Expiration
- Check if Anthropic has any credit expiration policies
- Some providers expire credits after a certain period
- Check your account terms for details

### Credit Transfers
- Credits are typically **not transferable** between accounts
- If you need to move credits, contact Anthropic support

### Multiple API Keys
- All API keys under the same account share the same balance
- Usage from any key deducts from the same account balance
- You can create multiple keys, they all use the same credits

---

## 🆘 If Credits Are Missing

### Check These:
1. **Account Login**: Make sure you're logged into the correct account
2. **Account Balance**: Check the billing section of your dashboard
3. **Credit History**: Review your transaction history
4. **Expiration**: Check if credits expired (if applicable)

### Contact Anthropic Support:
If your credits are genuinely missing:
- Email: support@anthropic.com
- Dashboard: Use the support/help section
- Explain: "My account balance is missing after API key deactivation"
